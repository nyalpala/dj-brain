# DJ Brain

DJ Brain is an AI-powered, local-first desktop application designed to help DJs organize music libraries, analyze tracks, build intelligent playlists, and prepare professional DJ sets.

---

## Features (Planned)

- Music Library Management
- Audio Analysis
- BPM Detection
- Musical Key Detection
- Camelot Wheel Support
- Smart Crates
- Playlist Builder
- AI Assistant
- Metadata Editing
- Offline-first Architecture

---

## Technology Stack

### Backend

- Python
- FastAPI
- SQLAlchemy
- PostgreSQL
- Alembic

### Frontend

- React
- TypeScript

- Tauri (Planned)
